# Script de Instalação de Aplicativos no Ubuntu

Este repositório contém um script `.sh` para facilitar a instalação de aplicativos no Ubuntu.

## Como usar

Clone o repositório:

```bash
git clone https://github.com/salesluis/auto-setup-os.git

```bash
cd auto-setup-os

```bash
chmod +x ./install.sh


```bash
sudo ./install.sh
